# Airbnb Clone 

This is an Airbnb Clone build with NextJS, Tailwind And MapBox.

Deployed it to the cloud with [Vercel](https://airbnb-clone-dusky-mu.vercel.app)

## Getting Started

Install the dependencies and run the development server

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
